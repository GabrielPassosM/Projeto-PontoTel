 ## Dependências utilizadas
No projeto, foram usadas, primordialmente, as seguintes dependências:
* [Python 3.8.](https://www.python.org/downloads/release/python-387/)
* [Flask](https://flask.palletsprojects.com/en/1.1.x/)
